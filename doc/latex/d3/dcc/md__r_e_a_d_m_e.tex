O \doxylink{class_sistema}{Sistema} da Locadora PDS é uma aplicação desenvolvida como trabalho final da disciplina de Programação e Desenvolvimento de Software II (PDS II) do curso de Sistemas de Informação na UFMG. Este projeto implementa um sistema abrangente de aluguel de filmes, proporcionando funcionalidades como cadastro de filmes e clientes, processos de locação, devolução de filmes e gerenciamento do catálogo.\hypertarget{md__r_e_a_d_m_e_autotoc_md3}{}\doxysection{\texorpdfstring{Equipe}{Equipe}}\label{md__r_e_a_d_m_e_autotoc_md3}

O \doxylink{class_sistema}{Sistema} da Locadora PDS visa otimizar as operações de uma locadora de filmes, oferecendo soluções para desafios comuns encontrados no gerenciamento de catálogos, controle de estoque, administração de clientes e operações de locação.\hypertarget{md__r_e_a_d_m_e_autotoc_md5}{}\doxysection{\texorpdfstring{Tecnologias Utilizadas}{Tecnologias Utilizadas}}\label{md__r_e_a_d_m_e_autotoc_md5}

\item {\bfseries{Linguagem de Programação\+:}} C++
\item {\bfseries{Compilação\+:}} Makefile
\item {\bfseries{Documentação\+:}} Doxygen
\item {\bfseries{Testes\+:}} Biblioteca doctest
\item {\bfseries{Controle de Versão\+:}} Git e Git\+Hub
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md6}{}\doxysection{\texorpdfstring{Estrutura do Projeto}{Estrutura do Projeto}}\label{md__r_e_a_d_m_e_autotoc_md6}
O projeto é organizado em classes, cada uma responsável por aspectos específicos do sistema\+:
\begin{DoxyItemize}
\item {\ttfamily formatacao}\+: Implementa métodos para formatação de strings.
\item {\ttfamily locacao}\+: Contém métodos e atributos relacionados às operações de locação.
\item {\ttfamily filme}\+: Classe base que engloba atributos e métodos comuns a todos os tipos de filmes.
\item {\ttfamily dvd}, {\ttfamily bluray}, {\ttfamily fita}\+: Classes que herdam de {\ttfamily filme} e implementam características específicas.
\item {\ttfamily cliente}\+: Classe que gerencia informações e operações relacionadas aos clientes.
\item {\ttfamily sistema}\+: Classe que administra operações do sistema, evitando sobrecarregar o arquivo principal.
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md7}{}\doxysection{\texorpdfstring{Funcionalidades Principais}{Funcionalidades Principais}}\label{md__r_e_a_d_m_e_autotoc_md7}

\begin{DoxyEnumerate}
\item {\bfseries{Cadastro de Filmes e Clientes\+:}}
\begin{DoxyItemize}
\item Permite o registro de novos filmes e clientes no sistema.
\end{DoxyItemize}
\item {\bfseries{Locação e Devolução de Filmes\+:}}
\begin{DoxyItemize}
\item Facilita o processo de aluguel e devolução de filmes, mantendo o controle do estoque.
\end{DoxyItemize}
\item {\bfseries{Listagem do Catálogo e Clientes\+:}}
\begin{DoxyItemize}
\item Apresenta informações detalhadas sobre os filmes disponíveis e a base de clientes.
\end{DoxyItemize}
\end{DoxyEnumerate}\hypertarget{md__r_e_a_d_m_e_autotoc_md8}{}\doxysection{\texorpdfstring{Funcionalidades Extras}{Funcionalidades Extras}}\label{md__r_e_a_d_m_e_autotoc_md8}
As seguintes funcionalidades extras foram implementadas\+:
\begin{DoxyItemize}
\item Leitura de arquivo pode ser utilizada para qualquer operação, não apenas cadastrar filme.
\item Implementação de pontuação de fidelidade, onde cada aluguel de filme vale 1 ponto e a cada 5 pontos adquiridos o cliente ganha um desconto de R\$10,00 na hora da devolução.
\item Implementação do aluguel de blurays.
\item Implementação do aluguel de aparelho leitor de bluray.
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md9}{}\doxysection{\texorpdfstring{Diferenças de input}{Diferenças de input}}\label{md__r_e_a_d_m_e_autotoc_md9}
Para conseguirmos trabalhar com nomes compostos para filmes, mudamos a ordem dos parâmetros nos inputs de Cadastrar \doxylink{class_filme}{Filme}. A ordem é\+:
\begin{DoxyItemize}
\item CF \texorpdfstring{$<$}{<}tipo D\texorpdfstring{$\vert$}{|}\+B\texorpdfstring{$\vert$}{|}F\texorpdfstring{$>$}{>} (dvd, bluray ou fita) $<$quantidade$>$ $<$codigo$>$ $<$categoria$>$ (para dvd e bluray) $<$nome\+\_\+do\+\_\+filme$>$ E quando se aluga um filme do tipo bluray, pode se adicionar o parâmetro ALUGAR\+\_\+\+APARELHO entre os parâmetros de categoria e nome do filme para solicitar o aluguel de um aparelho (que custa R\$20,00). O comando fica assim\+:
\item CF B $<$quantidade$>$ $<$codigo$>$ $<$categoria$>$ $<$\+ALUGAR\+\_\+\+APARELHO$>$ $<$nome\+\_\+do\+\_\+filme$>$
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md10}{}\doxysection{\texorpdfstring{Aprendizados}{Aprendizados}}\label{md__r_e_a_d_m_e_autotoc_md10}
