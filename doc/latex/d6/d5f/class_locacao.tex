\doxysubsection{Detailed Description}
Classe responsável por gerenciar as operações de locação de filmes. 

A classe \doxylink{class_locacao}{Locacao} realiza o cadastro, remoção, aluguel e devolução de filmes, além de manter registros dos filmes disponíveis no catálogo e dos clientes cadastrados.
