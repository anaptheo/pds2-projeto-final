O \doxylink{class_sistema}{Sistema} da Locadora PDS é uma aplicação desenvolvida como trabalho final da disciplina de Programação e Desenvolvimento de Software II (PDS II) do curso de Sistemas de Informação na UFMG. Este projeto implementa um sistema abrangente de aluguel de filmes, proporcionando funcionalidades como cadastro de filmes e clientes, processos de locação, devolução de filmes e gerenciamento do catálogo.\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Equipe}{Equipe}}\label{index_autotoc_md2}

O \doxylink{class_sistema}{Sistema} da Locadora PDS visa otimizar as operações de uma locadora de filmes, oferecendo soluções para desafios comuns encontrados no gerenciamento de catálogos, controle de estoque, administração de clientes e operações de locação.\hypertarget{index_autotoc_md4}{}\doxysection{\texorpdfstring{Tecnologias Utilizadas}{Tecnologias Utilizadas}}\label{index_autotoc_md4}

\end{DoxyItemize}\hypertarget{index_autotoc_md5}{}\doxysection{\texorpdfstring{Estrutura do Projeto}{Estrutura do Projeto}}\label{index_autotoc_md5}
O projeto é organizado em classes, cada uma responsável por aspectos específicos do sistema\+:
\begin{DoxyItemize}
\item {\ttfamily formatacao}\+: Implementa métodos para formatação de strings.
\item {\ttfamily locacao}\+: Contém métodos e atributos relacionados às operações de locação.
\item {\ttfamily filme}\+: Classe base que engloba atributos e métodos comuns a todos os tipos de filmes.
\item {\ttfamily dvd}, {\ttfamily bluray}, {\ttfamily fita}\+: Classes que herdam de {\ttfamily filme} e implementam características específicas.
\item {\ttfamily cliente}\+: Classe que gerencia informações e operações relacionadas aos clientes.
\item {\ttfamily sistema}\+: Classe que administra operações do sistema, evitando sobrecarregar o arquivo principal.
\end{DoxyItemize}\hypertarget{index_autotoc_md6}{}\doxysection{\texorpdfstring{Funcionalidades Principais}{Funcionalidades Principais}}\label{index_autotoc_md6}

\begin{DoxyEnumerate}
\item {\bfseries{Cadastro de Filmes e Clientes\+:}}
\begin{DoxyItemize}
\item Permite o registro de novos filmes e clientes no sistema.
\end{DoxyItemize}
\item {\bfseries{Locação e Devolução de Filmes\+:}}
\begin{DoxyItemize}
\item Facilita o processo de aluguel e devolução de filmes, mantendo o controle do estoque.
\end{DoxyItemize}
\item {\bfseries{Listagem do Catálogo e Clientes\+:}}
\begin{DoxyItemize}
\item Apresenta informações detalhadas sobre os filmes disponíveis e a base de clientes.
\end{DoxyItemize}
